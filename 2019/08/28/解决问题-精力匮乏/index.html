生活方式

  1. 规律生活，用习惯代替毅力
  2. 张弛有度，每周日剖心烛照加放松日加欺骗日
  3. 远离坏习惯，一时快乐，长久空虚，没有未来可以期待
  4. 控制欲望，对权力、美食、购物等贪欲，让人不思进取，沉溺不能自拔

呼吸法

  • 呼气，呼到不能呼为止，感受腹部收紧的感觉。（此处可停顿1-2S）
  • 一直收紧腹部肌肉，开始吸气
  • 感受腹部肌肉与吸入气体的对抗，吸到不能吸为止。（此处可停顿1-2S）
  • 保持一直收紧腹肌，呼气。
  • 重复以上呼吸
  • 每天进行3组， 每组10次呼吸。 组间半分钟。
