向量

发表于 2019-09-03 |
  • 极大线性无关组：一个向量组中可以代表该向量组中所有成员的一组向量
  • 代表的个数则为向量组的秩

向量及向量组的线性相关性

  • 线性组合：$某个向量P可以表示为：k_1a_1+k_2a_2+k_3a_3+…+k_ma_m​$,称为向量P为向量组$a_1,a_2…a_m的线性组合​$
  • 线性表出：其实上面的是线性表出的意思，组合就是多个向量组合下，不需要等于P
  • 线性相关：存在$k_1a_1+k_2a_2+k_3a_3+…+k_ma_m=0$，且k不全为0
  • 线性无关:不存在被
  • 单个非零向量，两个不成比例的向量均线性无关
  • 判别线性相关性的七大定理
    • 向量组线性相关的充要条件是向量组至少有一个向量可由其余n-1个向量线性表出
    • $\alpha_1,\alpha_2​$线性无关，$\beta_1,\alpha_1,\alpha_2​$线性相关，则$ \beta_1可由\alpha_1,\alpha_2线性表示，且表示法唯一​$
    • 如果向量组$\beta_1 ..\beta_t可由向量组\alpha_1,\alpha_s$线性表示，且t>s，则$\beta_1 ..\beta_t$线性相关[以少表多，多的相关]
    • 设m个n维向量$\alpha_1…\alpha_m​$,则向量组$\alpha_1…\alpha_m​$线性相关的充分必要条件是齐次线性方程组Ax=0有非零解[|A|=0]
    • 向量$\beta 可有向量组\alpha_1…\alpha_m$线性表出，$Ax=\beta$有 解，r(A) = r([A,$\beta$])
    • 如果$向量组\alpha_1…\alpha_m一部分向量线性相关，则整个向量组也线性相关​$
    • 如果一组$向量组\alpha_1…\alpha_m线性无关​$，那么把这些向量各任意添加m个分量所得到的新向量（n+m维)组$向量组\alpha_1^…\alpha_m^​$也线性无关，而如果$向量组\alpha_1…\alpha_m​$线性相关，则他们各去掉若干个分量所得到的新向量组也是线性相关的。

极大线性无关组，等价向量组，向量组的秩

  • 在向量组$\alpha_1…\alpha_s$中，若存在部分$\alpha_1…\alpha_m$满足：(1)$\alpha_1…\alpha_m$线性无关，(2)向量组中任意向量$\alpha_i$均可由$\alpha_1…\alpha_m$线性表出，则称向量组$\alpha_1…\alpha_m$是原向量组的极大线性无关组
  • 设两个向量组:（I）$\alpha_1…\alpha_m$ (II)$\beta_1…\beta_t$,若(I)中每个向量均可由(II)中线性表出，则称向量组(I)可有(II)线性表出，如果(I)(II)可以相互线性表出，则称(I)(II)等价向量组，记作$(I)\cong(II)$
  • 等价向量组满足
    • $(I)\cong(I)$
    • 若$(I)\cong(II),则(II)\cong(I)​$
    • 若$(I)\cong(II),(II)\cong(III)，则有(I) \cong (III)$
    • 向量组和它的极大线性无关组是等价向量组
  • 极大线性无关组所含向量的个数r称为向量组的秩
    • 矩阵的秩=行列式的秩=向量组的秩
    • 若A通过初等行变换得到B，则
      • A的行向量组和B的行向量组是等价向量组
      • A和B的任何相应部分列向量组具有相同的线性无关性
    • 设向量组$\alpha_1…\alpha_m$及向量组$\beta_t…\beta_s$均可由$\alpha_1…\alpha_m$线性表出，则$r(\beta_1..\beta_s) \leq r(\alpha_1…\alpha_m)$

向量空间

若$\xi_1,..\xi_n是n维向量空间R^n中的线性无关的有序向量组，则任一向量\alpha \in R^n$均可由$\xi_1…\xi_i$线性表出，称有序向量组$\xi_1,..\xi_n$ 是$R^n$的一个基，基向量的个数n称为向量空间的维数，而[$a_1,…,a_n$]称为向量a在基向量$\xi_1,..\xi_n$ 下的坐标，或称为$\alpha$的坐标行向量
