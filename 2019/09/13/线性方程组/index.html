齐次线性方程组

对于Ax = 0

  • 有解的条件
    • r（A）=n 时，向量组线性无关，方程组有唯一零解
    • r(A) < n时，方程组有非零解，且有n-r个线性无关解
  • 解的性质
    • $A \xi_1 = 0 A\xi_2=0 则A(k_1\xi_1+k_2\xi_2) = 0 其中k_1,k_2是任意常数$
  • 基础解系和解的结构
    • 基础解系：设$\xi1,\xi_2…\xi{n-r}$满足:one: 方程组Ax=0的解 :two:线性无关 :three:方程组Ax=0在任一解均可由$\xi1,\xi_2,\xi{n-r}$线性表出，则称$\xi1,\xi_2…\xi{n-r}$为Ax=0的基础解系
    • 通解：设$\xi1,\xi_2,\xi{n-r}是Ax=0的基础解系，则k1\xi_1+k_2\xi_2+k_3\xi_3+k{n-r}\xi{n-r}$ 是方程组Ax=0的通解，其中$k_1,k_2…k{n-r}$是任意常数
  • 求解方法与步骤
    • 化为阶梯型矩阵
    • 按列找出一个秩为r的子矩阵，则剩余列的位置的未知数即为自由变量
    • 按基础解系定义求出$\xi1,\xi_2…\xi{n-r}$，并写出通解

非齐次线性方程组

  • 有解的条件
    • 若$r(A) \ne r([ A,b]) (b不能由a_1,a_2,…a_n线性表出)$则方程组无解
    • 若$r(A) = r([A,b])=n(即a_1,a_2,a_n线性无关，a_1,a_2,a_n,b线性相关)​$ 则方程组有唯一解
    • 若$r(A) = r([A,b])=r<n,则方程组无穷多解$
  • 解的性质
    • 设$\eta_1,\eta_2,\eta$是非齐次线性方程组Ax=b的解，$\xi$是对应齐次线性方程组Ax=0的解，则:one: $\eta_1-\eta_2$是Ax=0的解 :two: $k\xi +n$是Ax=b的解
  • 求解的方法与步骤
    • 化为阶梯型矩阵，求出对应的齐次线性方程组的通解，再加上一个非齐次线性方程组的特解即是非齐次线性方程组的通解
