矩阵

发表于 2019-08-17

知识点

  • AE = A

转置矩阵

  • $(A^T)^T=A$
  • $(kA)^T = kA^T$
  • $(A+B)^T = A^T+B^T$
  • $(AB)^T = B^TA^T$

正交矩阵

  • 设A是n阶方阵，满足$A^TA = E$，则称A是正交矩阵
  • $A是正交矩阵 \Leftrightarrow A^TA = E \Leftrightarrow A^T = A^{-1} \Leftrightarrow A的行（列）向量是标准正交向量组$

伴随矩阵

将元素替换为其代数余子式

  • 性质
    • $AA^ = A^A = |A|E$

逆矩阵

  • A可逆的充分必要条件是 $|A| \ne 0 $时，当 $|A| \ne 0 $ 时，A可逆且$ A^{-1} = \frac{1}{|A|}A^*$

  • 性质

    • 若$k \ne 0,则(kA)^{-1} = \frac{1}{k}A^{-1}$
    • AB也可逆，且$(AB)^{-1}=B^{-1}A^{-1}$
    • $A^T也可逆，且(A^T)^{-1} =(A^{-1} )^T $
    • $|A^{-1}|=|A|^{-1}​$

  • 求逆矩阵的方法

    • 用伴随矩阵
    • 初等变换法（当年学的）
    • 定义法（估计用不上）
    • 将A分解成若干个可逆矩阵乘积，利用性质2
    • 分块矩阵的逆，若A,B均是可逆方阵则
